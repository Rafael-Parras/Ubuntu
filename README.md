# Projeto Web via Terminal

## Descrição
Projeto criado utilizando apenas o terminal do Ubuntu

## Comandos Utilizados
- pwd
- ls
- mkdir
- touch
- cp
- mv
- rm
- nano
- vim
- git init
- git add
- git commit
- git remote
- git branch
- git push

## Dificuldades
- Uso inicial do VIM
- Comandos de navegação no terminal
